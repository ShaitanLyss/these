## Rust CLI and main library

### [Clap](https://docs.rs/clap/latest/clap/)
CLI.

### [Serde](https://serde.rs/)
Data serialization and deserialization.

### [Tera](https://keats.github.io/tera/)
Template engine.

Purpose : Template-based code generation.

### [Schemars](https://graham.cool/schemars/)
JSON Schema documents generation.

Purpose : Generating a JSON schema describing the building blocks, for use by other applications.

### [ThisError](https://docs.rs/thiserror/latest/thiserror/)
Convenient way to create error types.
